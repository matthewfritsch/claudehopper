  <name>Task 1: Implement ShareFiles, PickFiles, UnshareFiles in internal/profile/share.go</name>
  <files>internal/profile/share.go, internal/profile/share_test.go</files>
  <behavior>
    - ShareFiles creates symlink in target profile pointing to resolved source file (EvalSymlinks to avoid chains)
    - ShareFiles adds path to target manifest shared_paths with source profile name as value
    - ShareFiles deduplicates: does not double-add to managed_paths
    - ShareFiles dry-run returns paths without filesystem changes
    - PickFiles copies file content (not symlink) from source to target profile
    - PickFiles preserves symlinks when source is symlink (os.Readlink + os.Symlink)
    - PickFiles handles directory sources with recursive copy preserving symlinks
    - PickFiles adds path to target manifest managed_paths (deduped)
    - UnshareFiles resolves symlink target, replaces symlink with real file copy
    - UnshareFiles removes path from shared_paths in manifest
    - UnshareFiles handles missing symlink targets gracefully (remove from shared_paths, skip copy)
    - All three accept explicit dir params (no internal config calls) for test isolation
  </behavior>
  <action>
    Add three new exported functions to the EXISTING `internal/profile/shared.go` file (which already contains DefaultLinked, SharedDir, copyFile, etc.). Also add a `copyDirRecursive` helper.

    **ShareFiles(profilesDir, srcName, tgtName string, paths []string, dryRun bool) ([]string, error):**
    - Validate both profile dirs exist
    - For each path: resolve source with `filepath.EvalSymlinks` to avoid symlink chains, then `renameio.Symlink(realTarget, dst)`
    - Update target manifest: `SharedPaths[path] = srcName`, add to ManagedPaths if not present (check with loop, not just append)
    - Return list of shared paths
    - If dryRun: collect paths but skip symlink creation and manifest save

    **PickFiles(profilesDir, srcName, tgtName string, paths []string, dryRun bool) ([]string, error):**
    - For each path: check source type with os.Lstat
      - If symlink: `os.Readlink(src)` then `os.Symlink(target, dst)` — preserve the symlink
      - If directory: `copyDirRecursive(src, dst)` — new helper
      - If regular file: `copyFile(src, dst)` — reuse existing helper
    - Update target manifest: add to ManagedPaths if not present (deduplicate)
    - Return list of picked paths
    - If dryRun: collect paths but skip copy and manifest save

    **UnshareFiles(profilesDir, tgtName string, paths []string, dryRun bool) ([]string, error):**
    - If paths is empty, unshare ALL shared_paths from the target manifest
    - For each path in shared_paths (or specified subset):
      - Build list of paths to unshare first (don't mutate while iterating)
      - Resolve symlink target with `filepath.EvalSymlinks`
      - If target exists: remove symlink, copy real file/dir in its place
      - If target does not exist: just remove symlink and skip copy (match Python behavior)
      - Remove from manifest SharedPaths
      - Add to ManagedPaths if not present
    - Save manifest
    - If dryRun: collect paths but skip filesystem changes and manifest save

    **copyDirRecursive(src, dst string) error:** (unexported helper)
    - `os.MkdirAll(dst, srcInfo.Mode())`
    - `os.ReadDir(src)` and iterate entries
    - For symlinks: `os.Readlink` + `os.Symlink` (preserve symlinks per Python shutil.copytree symlinks=True)
    - For dirs: recurse
    - For files: `copyFile(src, dst)`

    Write tests in NEW file `internal/profile/share_test.go` using t.TempDir():
    - TestShareFiles: create two profiles, share a file, verify symlink exists and manifest updated
    - TestShareFiles_ResolvesChainedSymlinks: source is a symlink, verify target is resolved (not chained)
    - TestShareFiles_DryRun: verify no filesystem changes
    - TestShareFiles_Dedup: share same file twice, verify managed_paths has no duplicates
    - TestPickFiles_RegularFile: copy file, verify content matches, verify managed_paths updated
    - TestPickFiles_Symlink: source is symlink, verify destination is also a symlink (preserved)
    - TestPickFiles_Directory: copy dir, verify recursive contents
    - TestUnshareFiles_ReplacesSymlink: create shared symlink, unshare, verify real file exists
    - TestUnshareFiles_ManifestCleaned: verify shared_paths entry removed after unshare
    - TestUnshareFiles_MissingTarget: symlink target deleted, verify graceful skip
    - TestUnshareFiles_AllPaths: empty paths arg unshares everything in shared_paths
