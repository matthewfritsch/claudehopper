    - Test: CreateFromProfile copies all files from source profile dir, writes created_from in manifest
    - Test: CreateFromProfile preserves symlinks as symlinks (os.Readlink + os.Symlink)
    - Test: LinkDefaultsIntoProfile creates shared dir if missing, seeds DEFAULT_LINKED files
    - Test: EnsureSharedDefaults creates ~/.config/claudehopper/shared/ if missing
    - Test: All three create functions call LinkDefaultsIntoProfile for default shared file bootstrapping

  <action>
1. Create `internal/profile/shared.go`:
   - `var DefaultLinked = []string{"settings.json", "settings.local.json", ".mcp.json"}` (matches Python DEFAULT_LINKED)
   - `func SharedDir() (string, error)` - returns `config.ConfigDir() + "/shared"`
   - `func EnsureSharedDefaults() (string, error)` - creates shared dir via `os.MkdirAll`, returns path
   - `func LinkDefaultsIntoProfile(profileDir string, fromSource string) error` - for each DEFAULT_LINKED file:
     - If file exists in shared dir: symlink from profileDir/file -> sharedDir/file
     - If file does NOT exist in shared dir but exists in fromSource (when non-empty): copy fromSource/file to sharedDir/file, then symlink
     - If file exists nowhere: skip (blank profile, no seed source)
     - Record in manifest shared_paths with value "(shared)"

2. Create `internal/profile/create.go`:
   - `type CreateOptions struct { Name, Description, FromProfile string; FromCurrent, Activate bool }`
   - `func CreateBlank(profilesDir, name, description string) error`:
     - Validate name, create dir, write empty settings.json, create manifest with managed_paths=["settings.json"], call LinkDefaultsIntoProfile with empty fromSource
   - `func CreateFromCurrent(profilesDir, claudeDir, name, description string) error`:
     - Validate name, create dir
     - Scan claudeDir with os.ReadDir, filter: skip IsProtected, skip `.hop-*` prefix, skip `.ccswap` prefix
     - For each entry: if symlink (os.Lstat), preserve via os.Readlink+os.Symlink and record in shared_paths with "(shared)"; if regular file/dir, copy to profile dir and add to managed_paths
     - Call LinkDefaultsIntoProfile with fromSource=profileDir (the newly created profile dir)
     - Save manifest
   - `func CreateFromProfile(profilesDir, sourceName, newName, description string) error`:
     - Validate newName, resolve source profile dir, verify source exists
     - Copy source dir recursively: use filepath.Walk + os.Lstat; for symlinks use os.Readlink+os.Symlink; for files use io.Copy; for dirs use os.MkdirAll
     - Load source manifest, set CreatedFrom=sourceName, update Description if provided
     - Call LinkDefaultsIntoProfile with fromSource=profileDir
     - Save manifest to new profile dir

   All functions accept explicit directory paths (not calling config.ProfilesDir internally) to keep them testable with t.TempDir().

3. Create `internal/profile/shared_test.go` with tests for EnsureSharedDefaults and LinkDefaultsIntoProfile using t.TempDir().

4. Create `internal/profile/create_test.go` with tests for all three create functions. Use t.TempDir() to create fake claudeDir and profilesDir. Verify:

   - Symlinks preserved as symlinks (check with os.Lstat + os.Readlink)
   - Protected/hop/ccswap files excluded from from-current capture

5. Create fixture files in `internal/profile/testdata/` for expected manifest outputs.
